# python_CLI_quiz_app
A simple command-line based quiz application built using Python.
## Description
This project is a Multiple Choice Question (MCQ) quiz that runs in the terminal.  
The user selects answers (A/B/C/D), and the program calculates the final score and percentage.

## Features
- Multiple choice questions
- Input validation (only A, B, C, D allowed)
- Score calculation
- Percentage calculation
- Clean loop-based structure using lists and dictionaries

## Concepts Used
- Python Lists
- Dictionaries
- For loops
- While loops
- Conditional statements (if-elif-else)
- Input validation
- Basic project structuring
